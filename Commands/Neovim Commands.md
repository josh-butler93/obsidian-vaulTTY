The 'space button' is the leader key that will open up a menu for you
- space

To open a new window
- Vertical:
	- command:
		- :Vex 
- Horizontal:
	- command:
		- :Hex 

To navigate between the windows
- Type:
	- control + k 
		- to go up
	- control + j
		- to go down

To exit out of the window
- command:
	- :q 
		- This will close your current window

To create a new Directory run the below command
- command:
  - !mkdir <direcotry_name>
